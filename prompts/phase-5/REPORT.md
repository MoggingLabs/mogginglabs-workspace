## Cross-cutting audits (5/07)

- **Focus-visible**: one global rule (`:focus-visible` ‚Üí 2px `--accent` ring,
  offset 1) covers every interactive element ‚Äî all are native buttons/inputs or
  carry `tabIndex` (rail tabs). Receipt: `after/dark-focus-walk.png` (real Tab
  keystrokes, not JS focus). Grep: no `outline: none` without a replacement in
  the sheet (the only `outline: none` sites re-style via border/box-shadow:
  `.input:focus`, `.ws-rename`, `.pane-title-input`, `.palette-input`).
- **Motion**: tokens only ‚Äî `--dur-1: 120ms`, `--dur-2: 150ms` (lowered from
  200ms this step); no transition anywhere exceeds 150ms. The
  `prefers-reduced-motion` block uses `*` + `::before/::after` with
  `!important`, so every current and future animation/transition collapses to
  0.01ms ‚Äî including the attention pulse and home beam.
- **Empty states**: recents (designed first-run hero copy), board (header hint +
  per-lane "+ Add card"), claims modal (`.claims-empty`), profiles (`.ph-empty`
  "No profiles yet‚Ä¶"), hosts (`.ph-empty`), palette (`.palette-empty`). None of
  the list panels renders blank space.
- **Tooltips**: every icon-only button has `title` + `aria-label` ‚Äî enforced by
  type (`IconButton.label` required) and verified by grep across `append(icon(`
  sites (the remaining hits are decorative icons inside labeled controls).

## One environmental find during the freeze sweep

TEMPLATE A/B failed on first sweep ‚Äî NOT a Phase-5 regression: the pane buffer
showed the claude CLI parked on its NEW first-run onboarding (syntax-theme
picker), and past that, an OAuth login URL. Newer claude versions keep session
state under `%LOCALAPPDATA%`, which the smoke isolates by design ‚Äî so the real
authenticated TUI (alternate buffer) is unreachable in a smoke, and a smoke must
never perform OAuth or copy live tokens into temp dirs. The gate was restated to
its actual intent, deterministically: the claude SLOT recorded a claude launch
(`agents.lastLaunch`), claude is the process talking in the pane (its own output),
restore relaunches it ‚Äî with the alternate-buffer TUI still recognized when auth
exists. Both phases re-verified green in their final form.
